# Setup
> Composer Installation
> [![Download]](https://getcomposer.org/download/) </br>
```bash
composer install
```
> Laravel Installation
```bash
php artisan migrate:fresh --seed
```
> VueJS Installation
```bash
npm install
```
Or
```bash
pnpm install
```

# Running Development
> Laravel
```bash
php artisan serve
```
> VueJS
```bash
npm install
```
Or
```bash
pnpm dev
```
